# Just-Like-Home
Description:
"Just Like Home" is a full-stack web application designed to assist international students in Canada. This app provides a user-friendly platform to discover local services that align with the students' cultural preferences, helping them feel more at home in a new country. It features a responsive and intuitive interface, backed by a robust Django server for data management and integrates Google Maps API for location-based services.

Features:
User-Friendly Interface: Developed using React.js and Material-UI, offering a seamless and responsive user experience.
Cultural Alignment: Database of local services relevant to the diverse cultural backgrounds of international students.
Location-Based Services: Integration of Google Maps API to help users find services based on their current location.
Robust Backend: Built with Django, ensuring efficient data storage, retrieval, and management.
